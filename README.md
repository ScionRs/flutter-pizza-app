# Pizza_app
Домашний проект для освоения навыков работы с Flutter по тематике пиццы :pizza:

## Основной функционал :
 - Поиск продуктов 
 - Множественный выбор ингредиентов
 - Выбор размера продукта
 - Система начисления бонусов
